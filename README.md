# Tip Calculator

A simple and user-friendly Tip Calculator built using HTML, CSS, and JavaScript. It helps users calculate the tip amount, total bill, and how much each person should pay when splitting the bill.


## 📌 Features
- Calculate tip based on percentage
- Split bill among multiple people
- Displays:
  - Total Tip
  - Total Amount
  - Amount per Person
- Clean and responsive UI
- Easy to use


## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript


## 🚀 How to Use

1. Open the project folder
2. Run the index.html file in your browser
3. Enter:
   - Bill Amount
   - Tip Percentage
   - Number of People
4. Click on Calculate
5. View the results instantly


## 📂 Project Structure

Tip-Calculator/ │── index.html │── style.css │── script.js


## ⚠️ Input Validation
- All fields must be filled
- Number of people must be greater than 0
- Shows error message for invalid input
